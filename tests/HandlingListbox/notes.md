
Handling Listbox or Dropdown
===========================
 in web applications, listbox/dropdown is mainly classified two types

 1. single-select listbox
 2. multi-select listbox

 Single-select listbox
 =====================
   
    used to select the only one option at a time

    To select value in the listbox we have selectoption()

    We can pass any of three properties
    
       1. index
       2. value
       3. label

Getting Selected option in dropdown
===================================
    We can use option:checked in Css selector Expression to get selected option in dropdown

Multi-select listbox
====================
  To Select values -> SelectOption([{index:0},{index:1},{label:'Paytm'}])
